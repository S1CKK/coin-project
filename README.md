# 🚀 Coin Project (Flutter)

A Flutter application that displays cryptocurrency data with search, pagination, and invite features.

---

## 📱 Features

- 📊 List of coins from API
- 🔍 Search coins
- ♻️ Infinite scroll (fetch more)
- 🎯 Invite Friends feature
- 🌐 Localization (multi-language support)

---

## 🛠 Tech Stack

- Flutter
- Dart
- Cubit / Bloc
- REST API
- flutter_gen (l10n)

---

## 🚀 Getting Started

### 1. Clone the repo

```bash
git clone git@github.com:S1CKK/coin-project.git
cd coin-project
```

### 2. Install dependencies
```bash
flutter pub get
```

### 3. Run the app
```bash
flutter run
```

## 📦 Build
### Android
```
flutter build apk
```

### iOS
```
flutter build ios
```
